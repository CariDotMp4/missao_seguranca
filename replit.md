# Missão Segurança - Jogo Educativo

## Visão Geral
Jogo digital educativo sobre Segurança do Trabalho e EPIs (Equipamentos de Proteção Individual), desenvolvido com HTML5, CSS3 e JavaScript puro.

## Status Atual
✅ **Projeto Completo e Funcional**
- Interface completa com 3 telas (Inicial, Quiz, Resultado)
- 20 perguntas sobre segurança do trabalho no banco de dados
- Sistema de pontuação e classificação por níveis
- Design responsivo com tema de segurança (amarelo, preto, verde)
- Servidor HTTP rodando na porta 5000

## Estrutura do Projeto

### Arquivos Principais
- `index.html` - Página principal com estrutura do jogo
- `style.css` - Estilos responsivos com tema de segurança
- `script.js` - Lógica do jogo, banco de perguntas e sistema de pontuação
- `README.md` - Documentação completa do projeto
- `assets/` - Pasta para imagens ilustrativas

### Características Técnicas
- **Frontend**: HTML5, CSS3, JavaScript Vanilla
- **Servidor**: Python HTTP Server (porta 5000)
- **Responsivo**: Adaptado para desktop, tablet e mobile
- **Sem dependências externas**: Executa direto no navegador

## Funcionalidades

### Mecânica do Jogo
1. **10 perguntas aleatórias** selecionadas de um banco de 20
2. **4 alternativas** por pergunta
3. **Sistema de feedback educativo** para cada resposta
4. **Pontuação**: 1 ponto por acerto
5. **Classificação final**:
   - 0-3 acertos: Nível Básico
   - 4-7 acertos: Nível Intermediário
   - 8-10 acertos: Nível Avançado

### Temas das Perguntas
- EPIs e seu uso correto
- Sinalização de segurança
- Prevenção de acidentes
- CIPA e normas de segurança
- Proteção contra riscos específicos
- Procedimentos de emergência

## Como Usar
1. Abrir o arquivo `index.html` diretamente no navegador, ou
2. Usar o servidor configurado (porta 5000)
3. Clicar em "Iniciar Jogo"
4. Responder as perguntas
5. Ver resultado e classificação
